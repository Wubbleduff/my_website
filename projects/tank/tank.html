Self-Driving Tank

The Project

For my Computer Engineering class (CS100), my friend and I programmed a self-driving tank to navigate an obstacle course. The tank was programmed using ARM assembly. It used a state machine driven by infrared sensors to control the speed of each tank wheel. The wheels speed was controlled using Pulse-Width Modulation.
